QQ技术群，微信技术群，付费圈

作者:泰戈尔    发布于: 2019-05-26

在我刚学编程的时候，总是会热衷于加入一些QQ技术群，微信群。当然，为什么不加入那些付费群呢？因为没钱嘛。

但是动辄成百上千的群成员真的让人头大，有些活跃的群只需要五分钟就可以把未读消息弄到999+，有些不活跃的群，你发了一个精心描述的技术问题，结果石沉大海。这两种都属于比较极端的case，但是很明显都是不适合学习东西的。为什么这么说，可以自己体会下。如果这篇文章恰好被不认同此观点的你看到了，那也无所谓，反正这儿也没法评论，哈哈哈。我的底盘我做主，俺想咋写就咋写。

说完了不适合学习东西的，那么有没有适合的呢？

我个人觉得，有一定门槛才能进入的圈子，才是一个比较理想的状态。因为有门槛，所以人数上不会有很多。而且从相互的经历上来看，都有所接近。好比结对编程，能结对的基本上都是水平差不多的，如果一个水平高，一个水平低，那就不叫结对编程了，最后肯定都是会变成一个人领着一个人在开发。这就失去了结对编程的最初的意义。

有门槛，是不是付费就属于有门槛呢？我觉得，算是吧。但是和我想象中的门槛定义不是很一样。我想法里的门槛是：技术，智商，情商，经历等等上的，而不仅仅是物质上。物质上可以用💰很快就补上了，但是弥补不了交流的深度，对自身的提高就有了限制。

… …

不明所以的写了不少，最后总结下，我个人的审查态度就是：

看看质量到底怎么样？人多吗，成员的水平分布怎么样？活跃吗？能通过交流获取新的灵感或者领悟吗？
