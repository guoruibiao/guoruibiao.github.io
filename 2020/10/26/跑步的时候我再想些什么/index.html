跑步的时候我在想些什么

作者:泰戈尔    发布于: 2020-10-26

中道崩殂的思路

从大二开始，唯一坚持下来的就是跑步了。正因为如此，工作近一年了，体重没有明显的增加，也算是对自己一个交代。

之前看村上春树的《跑步的时候我在想些什么》那本书，虽然没看完，但是和村上不一样的是，我甚至没有对跑步有任何概念，对我来说，就是抬腿，踏脚，一点点跑完预定的路线。

这半年一直在芍药居地铁站附近跑，绕一圈大概2.63公里，说实话有点少。相较于之前在大连，围着操场跑体验很不一样，北京的地面并不平整，还得时刻注意着来往的车辆，一会就分心了，所以基本上什么也想不完。

大部分都是开了个头，然后就结束了。手机上有个TODOlist的APP，每次有啥好点子我都会迅速记录下。根据紧急、重要程度，分了四个维度，分别是：

  • 重要紧急
  • 重要不紧急
  • 不重要但紧急
  • 不重要不紧急

感觉还蛮有用，后面也要坚持下来。

重复造轮子值不值

我这个人思路很奇特，脑子里总会有一些乱七八糟的奇奇怪怪的小点子，而且总想着用代码去实现。

  • 看到Python有个commands的库 -> 最近在学golang -> 我得去实现一个类似的。
  • 同事转发了一个“杨超越鼓励师”的VS Code插件 -> 写博客想实时统计写了多少字 -> 我也得写一个Word count的VS Code 插件。
  • 某些知识点遗忘的忒快 -> 听说艾宾浩斯遗忘曲线可以帮助记忆 -> 写个程序，以时间线为轴，到时见就提醒我复习XX知识点。
  • … …

回想起来，自从大一下开始学Java以来，类似的想法就没断过，我总是会有这些奇奇怪怪的想法，而且总想着用代码实现实现。遗憾的是，很多东西到最后做的并不让人满意，至少我觉得它们没达到我脑海里的预期效果。

互联网上从来就不缺乏拥有相同想法的人，也不缺乏去实践的人，而我的很多想法，其实已经被很多人实现过了，而且看着还不赖，我就反思我的这种重复造轮子的行为值不值。或许是“不到黄河心不死”， 总想去试一试。今后如果再有类似的想法，还是得先斟酌斟酌，时间有限，不能老是浪费在某些无意义的事情上。

“娱乐至死”？

最近发现自己花费在手机上的时间越来越多了，这不是一个好现象，现在的放纵，未来肯定会后悔的。忘记是哪一本书上说的了：

当你有两个选择，一个是可以获得短暂快乐的娱乐，一个是枯燥乏味的学习。一定要选择后者，选择越困难，未来收获的快乐也就越多。

想想也是。我现在打王者荣耀打得再好能怎样，能靠这个养活自己吗？好像并不能。2019年05月19日00:00:26的我，手放在键盘上，突然想到，当初买这个MBP的时候，是要好好使用它的，结果现在下了班之后并没有完成这个预期。有点 -_-|…

最开始打游戏是为了解决无聊，但是打了游戏之后仿佛更加的无聊了。那为什么还要花费时间打游戏？我得好好想想了。

适度娱乐，不时想想，手上还有没有优先级更高的事情，做完之后，再娱乐呗。话说回来，我好像除了跑步，没有其他才艺了。看身边的人会唱歌的唱歌，会弹吉他的弹吉他，我仿佛是个nerd。现在或许没啥，等以后年龄大了，估计连广场舞初审都过不了，只能在旁边呆呆的看着了吧。

我得培养一项新的技能，具体是啥我也不知道，**TODO**吧。
